# Personal Portfolio – WordPress Theme

A custom WordPress theme built to showcase my portfolio, technical skills, and projects. The theme is designed with a clean, modern UI and structured to highlight both development and design capabilities.

## 🚀 Features

* Fully custom WordPress theme
* Dynamic skills section (PHP-powered)
* Responsive design for all screen sizes
* Organized portfolio and project display
* Clean and minimal UI/UX
* Easily customizable through WordPress

## 📦 Installation

1. Clone the repository:

```bash id="q1m9cx"
git clone https://github.com/your-username/your-repo.git
```

2. Move the theme to your WordPress installation:

```id="4n4o2v"
/wp-content/themes/your-theme-name
```

3. Activate the theme:

* Go to **WordPress Dashboard → Appearance → Themes**
* Select and activate the theme

## ▶️ Usage

* Manage content through the WordPress admin panel
* Customize layouts via theme files
* Modify styles in `style.css` or SCSS files
* Extend functionality using `functions.php`
